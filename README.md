# Airbnb Price Prediction (Machine Learning Project)

## Project Overview
This project applies supervised machine learning to predict Airbnb listing prices across European cities.

The goal is to evaluate different models, understand the impact of preprocessing, and identify key factors influencing pricing.

---

## Dataset
- Source: Kaggle Airbnb Europe dataset  
- Link: https://www.kaggle.com/datasets/jasonairroi/airbnb-market-data-europe
- Task: Regression  
- Target variable: `booked_rate_avg`

---

## Research Questions
1. Baseline performance of simple models  
2. Comparison of different machine learning models  
3. Effect of preprocessing techniques  
4. Feature importance and interpretability  
5. Sensitivity to evaluation metrics  
6. Robustness and generalization  
7. Practical usefulness for pricing strategy  

---

## Models Used
- Linear Regression  
- Ridge Regression  
- Decision Tree Regressor  
- Random Forest Regressor  
- Gradient Boosting Regressor  

---

## Evaluation Metrics
- MAE (Mean Absolute Error)  
- RMSE (Root Mean Squared Error)  
- R² Score  

---

## How to Run
1. Download the dataset from the link above  
2. Place dataset files in the same folder as the notebook  
3. Open the notebook:
   airbnb_all_research_questions_FIXED.ipynb
4. Run all cells
